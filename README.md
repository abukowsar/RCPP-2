# RCPP – Roadmap for Career & Project Planning

RCPP is a web-based platform designed to help students and learners plan their career paths, skill development, and project journeys in a structured and intuitive way. The platform focuses on guiding learners toward future-ready skills through clear roadmaps and practical planning tools.

Live Website: https://rcpp.vercel.app/

---

## Overview

Many students struggle to decide what to learn next and how to prepare for their desired careers. RCPP addresses this challenge by offering a clean and easy-to-use platform where learners can explore structured career and project roadmaps. The goal is to simplify decision-making and encourage focused, goal-oriented learning.

---

## Installation & Setup

1. Clone the repository  
   git clone https://github.com/your-username/rcpp.git  

2. Navigate to the project directory  
   cd rcpp  

3. Install dependencies  
   npm install  

4. Start the development server  
   npm run dev  

The application will run at http://localhost:3000
